Tips Command Line Linux Lanjutan — 430+ Perintah Lengkap

Halaman ini berisi kumpulan lebih dari 430 perintah Linux lengkap dengan penjelasan singkat agar kamu bisa kuasai command line sampai tingkat mahir.

1. Navigasi dan Manipulasi File (50+ Perintah)


ls -alh                        # List semua file dengan detail dan ukuran mudah dibaca
cd /path/to/dir               # Pindah direktori
pwd                           # Cetak direktori saat ini
mkdir -p dir/subdir           # Buat direktori bersarang
rm -rf dir/                  # Hapus direktori rekursif tanpa konfirmasi
cp -r source dest            # Copy direktori secara rekursif
mv file1 file2               # Rename/move file
find / -name 'file.txt'      # Cari file dari root
locate file.txt              # Cari file cepat (update database wajib)
stat file.txt                # Info detail file (ukuran, permission, dll)
file file.txt                # Tampilkan tipe file (biner, text, dll)
basename /path/to/file.txt   # Ambil nama file saja
dirname /path/to/file.txt    # Ambil direktori file
readlink -f file             # Resolve symbolic link absolut
tree -L 2                    # Tampilkan struktur direktori (harus install)
lsattr file                 # Tampilkan atribut khusus file (immutable, append-only)
chattr +i file              # Set file immutable (tidak bisa diubah)
touch file.txt              # Buat file baru atau update timestamp
split -b 10M bigfile.bin part # Pecah file jadi 10MB
cat file1 file2 > file3     # Gabung file
head -n 20 file.txt         # Tampilkan 20 baris pertama file
tail -n 50 file.txt         # Tampilkan 50 baris terakhir file
tail -f /var/log/syslog     # Live update file log
wc -l file.txt              # Hitung jumlah baris file
diff file1 file2            # Bandingkan isi file
cmp file1 file2             # Bandingkan file byte per byte
md5sum file                 # Hitung hash MD5
sha256sum file              # Hitung hash SHA256
strings binaryfile          # Tampilkan string teks dalam file biner
gzip file                  # Kompres file
gunzip file.gz             # Ekstrak file gzip
zip archive.zip file1 file2 # Buat arsip zip
unzip archive.zip           # Ekstrak zip
ln -s /path/to/file linkname # Buat symbolic link
ls -i                      # Tampilkan inode file
filefrag file               # Lihat fragmen disk file

2. Permission & Ownership (40+ Perintah)


chmod 755 file.sh           # Set permission rwxr-xr-x
chmod u+s file              # Set SUID bit (jalankan dengan hak owner)
chmod g+s dir               # Set SGID bit (inherit group untuk file baru)
chmod o+t dir               # Set sticky bit (hanya owner yang bisa hapus)
stat file                  # Lihat detail permission dan owner
chown user:group file       # Ganti owner dan group
chown -R user:group dir     # Ganti owner dan group rekursif
groups username             # Lihat grup user
id username                 # Info UID, GID, grup user
getfacl file                # Tampilkan ACL permission lanjutan
setfacl -m u:user:rwx file  # Tambah ACL permission
setfacl -x u:user file      # Hapus ACL user
umask                      # Tampilkan default permission saat buat file
umask 022                  # Set default permission (rw-r--r--)
lsattr file                # Tampilkan atribut file
chattr +i file             # Set immutable
chattr -i file             # Hapus immutable
chattr +a file             # Append-only
chattr -a file             # Hapus append-only

command &> alloutput.txt      # Redirect stdout dan stderr

  echo "This is a function"
